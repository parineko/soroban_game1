# 🧮 そろばん かんたん ゲーム ✨

小学生向けの「そろばん読み取りゲーム」＆「学習用シミュレーター」を Astro で実装した Web アプリです。
SVGによるリアルな描画と、直感的なタッチ操作に対応しています。

## 🌟 特徴

- **🌸 かわいいデザイン**: パステルカラーと丸みのあるUIで、お子様が楽しく遊べます。
- **📱 スマホ・PC完全対応**: SVG描画により、どんな画面サイズでも美しく表示されます。
- **👆 タッチ操作**: シミュレーターモードでは、本物のそろばんのように指で玉を弾いて動かせます。
- **⚙️ 自由な難易度設定**: 「桁数（1〜5桁）」と「表示時間（0.5〜3秒）」を自由に組み合わせて特訓できます。

---

## 🎮 ゲームモードと機能

### 1. 読み取りゲーム（本編）
画面に一瞬だけ表示されるそろばんの数字を読み取って答えるゲームです。

* **流れ**:
    1.  **設定**: 桁数（1〜5桁）と 秒数（3.0〜0.5秒）を選択してスタート！
    2.  **出題**: 設定した秒数だけ、そろばんが表示されます。
    3.  **回答**: テンキーで数字を入力。「つぎへ」で次の問題へ。
    4.  **結果**: 10問終了後、正解数とスコアを表示します。

### 2. そろばんシミュレーター
自由に触って学べる練習モードです。

* **双方向操作**:
    * **玉をタッチ**: 指で玉を動かすと、下の数字が自動で変わります。
    * **数字を入力**: テンキーで数字を入れると、玉が自動でパチパチ動きます。
* **視覚的な理解**: 「5の玉」と「1の玉」の仕組みを直感的に理解できます。
