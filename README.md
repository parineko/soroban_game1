# 🧮 そろばん かんたん ゲーム ✨

小学生向けの「そろばん読み取りゲーム」＆「学習用シミュレーター」です。  
Astro と SVG技術を使い、スマートフォンでもPCでも、本物のそろばんのような操作感を実現しました。

## 🌟 特徴

- **🌸 かわいいデザイン**: クリーム色やパステルカラーを基調とした、お子様が親しみやすいUI。
- **📱 マルチデバイス対応**: SVGによる動的描画で、画面サイズに関わらず美しく表示されます。
- **⚙️ 自由なカスタマイズ**: 桁数（1〜5桁）や表示速度（0.5秒〜3秒）を自由に設定して遊べます。
- **👆 リアルなシミュレーター**: 指で玉を弾いて計算できる「シミュレーターモード」を搭載。

---

## 🎮 ゲームモード

### 1. 読み取りゲーム（本編）
一瞬だけ表示されるそろばんの数字を読み取って答える、フラッシュ暗算の入門ゲームです。

* **流れ**:
    1.  **設定**: 「桁数」と「表示秒数」を選んでスタート！
    2.  **出題**: 設定した秒数だけ、そろばんが表示されます。
    3.  **回答**: テンキーで数字を入力し、「つぎへ」で進みます。
    4.  **結果**: 10問終了後、スコアと応援メッセージが表示されます。

### 2. そろばんシミュレーター
自由に触って学べる練習モードです。

* **双方向連動**:
    * **玉をタッチ**: 指で玉を動かすと、下の数字が自動で変わります。
    * **数字を入力**: テンキーで数字を入れると、玉が自動で動きます。
* **仕組みの理解**: 「5の玉」と「1の玉」の関係性を、手を動かしながら直感的に学べます。
